# Nellfs Blog

Personal blog built with [Hugo](https://gohugo.io/) using the [hugo-xmin](https://github.com/yihui/hugo-xmin) theme.

## Getting started

Clone the repo with submodules:

```bash
git clone --recurse-submodules <repo-url>
```

If you already cloned without submodules, initialize the theme:

```bash
git submodule update --init --recursive
```

Then build the site:

```bash
hugo
```

Or serve it locally with live reload:

```bash
hugo server
```

The site will be available at `http://localhost:1313`.
